# 中文词语填空测试生成项目 - 完整讨论总结

## 项目背景

为中文教学（IB/Cambridge框架 MYP IBDP IGCSE）创建一套**标准化、可复用的词语填空测试生成工具**，用于课文词汇教学。

## 最终产出物

### 三个核心文件

#### 1. 词语精选表（供教师选词）
文件：`[课文编号]_词语精选表(35选25).docx`
- 从课文筛选35个词语
- Word表格格式，横向排版（landscape）
- 列：序号、词语、含义、易错点（红色）、类别、难度（⭐）、选择框（☐）
- 难度分级：最难⭐⭐⭐⭐⭐（金色背景）、中等⭐⭐⭐（绿色背景）、容易⭐⭐（橙色背景）
- 教师勾选确认25个词后，进行第二步

#### 2. 学生填空测试卷（含教师答案卡）
文件：`[课文编号]_词语填空测试(打印版).docx`
- **三页一体的Word文档**：第一张纸双面打印（第一、二页），第二张纸为教师答案卡（第三页）
- **版面结构**（最终定版）：
  - 第一页正面：标题（课文《[名称]》）+ 副标题 + 学生信息 + 作答说明 + 评分说明（加粗：一共25题，每题4分，第19,20题每空2分）+ 题1-A（约40-45%题目）+ 剪裁线 + 拼音词库（5列×5行） + 剪裁线 + 题A+1-N
  - 第一页背面（第二页）：题N+1-25（剩余题目）
  - 第三页（独立纸）：教师答案卡（5列×5行答案表 + 难度分布说明）

#### 3. 内容备份（纯文本存档）
文件：`[课文编号]_词语填空测试_内容备份.md`
- 25个词汇表（按难度排列：最难10题 + 中等8题 + 容易7题）
- 25道填空题目（题号顺序固定，含标准答案）
- **打乱顺序的拼音词库**（单独列表，标注"⚠️ 与题目顺序故意不一致"）

## 核心设计要点

### 1. 题目顺序 vs 词库顺序
- **题目**：始终保持1-25题顺序不变
- **词库**：每次生成时**随机打乱**，与题目答案顺序**故意不一致**
- 目的：学生必须根据上下文判断，不能照顺序直接抄写
- 验证规则：词库第1个拼音不得是题1的答案，至少前5题无法直接对应

### 2. 词库位置（最终方案）
- **放在第一页题目中间**（而非页面底部），上下各一条剪裁线
- 词库上方：约40-45%的题目（题1-A）
- 词库下方：剩余题目继续排列（题A+1-N），跨页到第二页
- **撕下词库卡后**：两侧题目都完整保留，学生可继续作答

### 3. 两张纸的双面打印结构
```
第一张纸：
  正面（第一页）= 题1-A + 剪裁线 + 词库 + 剪裁线 + 题A+1-N
  背面（第二页）= 题N+1-25

第二张纸：
  正面（第三页）= 教师答案卡

撕下词库卡后：
  词库卡正面 = 拼音词库
  词库卡背面 = 第一页背面（第二页前段），保持干净，无题目
```

### 4. 评分与难度设置
- 总分：100分（25题 × 4分/题）
- 第19、20题：各有两个空，每空2分，共4分
- 4字词语（肃然起敬、深仇大恨）：必须完全正确，漏字或错字不得分
- **难度分布**：
  - 最难（40分）：睿智、肃然起敬、轻蔑、讽刺、嘲笑、暴虐、启蒙、诅咒、呐喊、虐待
  - 中等（32分）：宽恕、深仇大恨、批判、揭露、专制、祝福、污秽、腐败
  - 容易（28分）：战胜、武器、摧毁、照亮、陨落、纪念、平衡
- 评分建议：85分优秀 / 70-84分良好 / 60-69分及格 / 60分以下需补习

## 技术实现规范

### 使用工具
- **npm package `docx`**（docx-js）生成.docx文件
- **不用PDF**：拼音声调符号在服务器字体渲染困难，易乱码，Word格式最稳定

### 字体与样式
- 中文：SimSun
- 拼音与数字：Arial
- 标题：SimSun 加粗
- 题号：Arial 加粗
- 拼音词库：Arial 加粗，size 20，红色（C00000）
- 拼音词库背景：浅灰色（F5F5F5）

### 页面设置
```javascript
page: {
  size: { width: 12240, height: 15840 },  // US Letter (标准 8.5" × 11")
  margin: { top: 720, right: 900, bottom: 720, left: 900 }  // pt
}
```

### 关键高度计算
- 每题单行高度约：500-520 DXA
- 拼音词库（5行×5列）：约1700-1800 DXA
- 剪裁线：约280 DXA

## 关键经验教训

1. **版面设计的迭代**：
   - 初期方案（词库在页面底部）→ 最终方案（词库在题目中间）
   - 原因：词库在中间撕下后，两侧题目都保持完整，更实用

2. **词库随机的重要性**：
   - 必须严格保证词库与题目顺序不同
   - 防止学生照顺序直接抄，无法真正理解

3. **字体问题**：
   - PDF + 拼音声调 = 乱码地狱
   - Word格式最稳定，用户可自由编辑调整

4. **内容备份的独立性**：
   - md文件中词库单独列表，不与词汇总表混淆
   - 明确标注词库是打乱顺序的
