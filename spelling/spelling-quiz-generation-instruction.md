# 中文词语填空测试生成指令

## 使用方法
将此文件与课文内容同时提供给 Claude，Claude 将自动完成从词汇筛选到最终 Word 文档的全部流程，无需额外解释。

---

## 第一步：从课文中筛选35个词语

从课文中筛选35个词语，供教师最终选出25个，筛选标准：

1. **对写作有帮助**：适合在议论文、思想批判类文章中使用的词汇
2. **对阅读理解有帮助**：是课文关键概念和核心思想的载体词汇
3. **容易书写出错**：笔画复杂、易混淆、易漏笔的汉字

输出格式为 Word 表格，包含以下列：
- 序号
- 词语
- 含义
- 易错点（红色字体标注）
- 类别
- 难度（⭐数量）
- 选择框（☐）

难度分级：
- ⭐⭐⭐⭐⭐ 最难（笔画极复杂或极易误写）→ 金色背景
- ⭐⭐⭐ 中等 → 绿色背景
- ⭐⭐ 相对容易 → 橙色背景

表格横向排版（landscape），教师在选择框中勾选25个词。

---

## 第二步：生成填空测试 Word 文档

教师确认25个词后，生成打印版测试文档。

### 文档结构（共三页）

#### 第一页（第一张纸正面）
- 标题格式："课文《[课文名称]》"（居中，深蓝色，加粗）
- 副标题"词语应用填空测试"（居中）
- 学生信息栏：姓名 / 班级 / 日期
- 作答说明：提示学生参照拼音词库
- **评分说明（加粗）**：一共25题，每题4分，第19,20题每空2分（紧接作答说明下方单独一行）
- **第 1 - A 题**（前段题目，约总数的40-45%）
- 剪裁线（一行，灰色）：
  ```
  - - - - - - - - - - - - - - -  沿此线剪开，将拼音词库卡撕下备用  - - - - - - - - - - - - - - -
  ```
- 拼音词库表格（紧接第一条剪裁线下方）
- 剪裁线（第二条，与第一条完全相同，紧接词库下方）
- **第 A+1 - N 题**（中段题目，填满第一页剩余空间）

#### 第二页（第一张纸背面）
- **第 N+1 - 25 题**（剩余题目，填满第二页）

#### 第三页（独立第二张纸）
- 教师答案卡
- 评分说明
- 标准答案表格（5列×5行）
- 难度分布说明

### 题目与词库的位置逻辑
词库夹在题目中间（而非放在页面底部），两侧都有题目：
- 词库上方：约40-45%的题目（第1-A题）
- 词库下方：其余题目继续排列，跨页到第二页

这样设计的好处：学生撕下词库卡后，上下两段题目都不受影响，可以继续作答。

词库的位置由内容自然决定：在第一页中，前段题目结束后紧接剪裁线和词库，词库后继续剩余题目，第二页接续。

### 拼音词库设计
- **顺序随机打乱**（与题目顺序无关，增加答题难度）
- **严格禁止**词库顺序与题目顺序一致——学生不能照顺序抄写，必须真正根据上下文判断
- 生成后须验证：词库第1个拼音对应的词，不得是题目第1题的答案；以此类推，至少保证前5题无法直接对应
- 5列排列，每格只有拼音，无序号无标题
- 浅灰色背景，红色粗体拼音，带声调符号

### 剪裁线使用逻辑（双面打印）
```
第一张纸正面（第一页）：
  题1-A
  ── 剪裁线 ──
  拼音词库
  ── 剪裁线 ──
  题A+1-N

第一张纸背面（第二页）：
  题N+1-25

沿两条剪裁线撕下词库卡：
  词库卡正面 = 拼音词库
  词库卡背面 = 空白（第二页题目在另一面，不受影响）

第二张纸（第三页）：
  教师答案卡
```

## 第三步：技术实现规范

### 使用工具
- **docx-js**（npm package `docx`）生成 `.docx` 文件
- 输出路径：`/mnt/user-data/outputs/`

### 字体规范
- 中文内容：`font: "SimSun"`
- 拼音及数字：`font: "Arial"`
- 标题：`font: "SimSun"`，加粗

### 页面设置
```javascript
page: {
  size: { width: 12240, height: 15840 },  // US Letter
  margin: { top: 720, right: 900, bottom: 720, left: 900 }
}
```

### 关键样式
- 题目字号：`size: 24`（12pt），行间距：`line: 360`，段后：`after: 160`
- 题号：Arial 加粗
- 句子：SimSun 常规
- 剪裁线：Arial，`size: 16`，`color: "999999"`，居中
- 词库拼音：Arial 加粗，`size: 20`，`color: "C00000"`
- 词库格背景：`fill: "F5F5F5"`

### 答案表格样式
- 表头：深蓝色背景（`1F4E78`），白色文字
- 数据行：交替白色 / 浅蓝色（`EEF3FF`）
- 题号列：Arial 加粗
- 答案列：SimSun 加粗，红色（`C00000`）

## 第四步：同步生成内容备份 md 文件

同时生成一个纯文本 `.md` 文件，保存以下内容：

```markdown
# [课文标题] 词语填空测试内容备份

## 25个目标词汇
（序号、词语、拼音、含义、难度）

## 填空题目（1-25题，含答案）
（题号、句子、答案词语）

## 拼音词库（打乱顺序）
（按测试中实际顺序列出）
```

## 注意事项

1. **不要用 PDF**：拼音声调符号（ā á ǎ à 等）在服务器可用字体中渲染困难，易出现乱码。坚持使用 Word（.docx）格式，用户可自行编辑调整。

2. **词库位置**：词库夹在第一页题目中间，不推到页面底部。词库上方放约40-45%的题目，下方继续剩余题目。两条剪裁线夹住词库，撕下后两侧题目完整保留。

3. **词库必须完整在第一页**：词库和两条剪裁线必须都在第一页内，不能跨页。如空间不足，减少词库上方的题目数量。

4. **随机打乱**：每次生成时词库顺序重新随机，题目顺序始终保持 1-25。

5. **词库与题目顺序必须不同**：内容备份 md 和测试卷中，词库的拼音顺序必须与题目答案顺序不同。生成后检查：如果词库第N个拼音恰好对应题目第N题的答案，必须重新打乱，直到两者顺序明显不同为止。内容备份的词库表格须单独列出（不与词汇总表合并），并标注"⚠️ 与题目顺序故意不一致"。

6. **教师答案卡**：始终放在第三页（独立打印），答案按题目原始顺序（1-25）排列。
