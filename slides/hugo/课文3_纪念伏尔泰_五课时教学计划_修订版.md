# 课文3《纪念伏尔泰逝世一百周年的演说》五课时教学计划 修订版

## 修订说明

原计划的优点是路线清楚，能从“读懂、读深、会写、会用”逐步推进，词汇、朗读、写作和现实反思都有安排。修订时保留这一结构，同时补强五个更能体现本文文学学习价值的重点：

- “微笑，就是伏尔泰”：理解微笑作为理性、仁慈、讽刺和宽容的精神象征。
- “一枝笔”：理解思想如何成为反抗专制、迷信和不公的武器。
- “光明/黑夜/黎明/坟墓”：梳理全文的意象链，体会雨果将抽象思想视觉化的能力。
- “演说现场”：关注鼓掌、喝彩、呼告、排比和反复，体会演说文体的现场召唤力。
- “理性批判的边界”：引导学生理解批判不是情绪宣泄，而是基于证据、理性和责任的判断。

## 总体设计思路

本文是一篇议论性演说，核心价值不只是“敢说真话”，而是思想自由、理性批判、人道主义和宽容精神。五节课从“读懂 -> 读深 -> 会写 -> 会用”递进：第一课建立文本地图，第二课进入语言和修辞，第三课转化为议论文写作，第四课连接现实中的判断与表达，第五课完成词汇考核、写作定稿和单元总结。

词汇贯穿全程，但不孤立讲解；学生需要在朗读、细读、讨论和写作中反复使用。写作训练以课文为支架，强调“观点 + 事实 + 分析 + 修辞气势”。现实反思环节要避免把文本简化成“勇敢表达”，而要引导学生思考：什么时候应该批判，凭什么批判，怎样批判才有理性和责任。

## 第一课：读懂课文，建立基础

### 教学目标

- 激活背景知识，理解伏尔泰、雨果与启蒙精神的基本关联。
- 扫除核心词汇障碍，初步理解文章大意。
- 建立全文的五点文本地图：历史定位、社会批判、斗争方式、精神核心、高潮召唤。

### 教学流程

1. 导入：说真话的代价 10分钟

提问：“你有没有说过一句话，因为太真实而惹了麻烦，或者让别人开始认真思考？”  
教师引导学生把经验暂时放在“表达的代价”上，再补充一句关键区分：本文讨论的不只是个人勇敢，而是思想、良知和理性如何面对不公。

2. 背景进入 8分钟

简要说明：

- 伏尔泰：启蒙思想家，反对专制、迷信和司法不公。
- 雨果：人道主义作家，在伏尔泰逝世一百周年纪念会上发表演说。
- 演说情境：面对听众，既纪念死者，也召唤活人。

3. 词汇预习检查 15分钟

学生两两互测目标词。教师重点讲解抽象词和价值判断词，例如：

- 睿智、启蒙、批判、理性、公正
- 肃然起敬、轻蔑、讽刺、嘲笑、宽恕
- 专制、暴虐、迷信、不宽容、压迫

要求学生做到三步：会读、会解释、能放进句子里。

4. 课文朗读与初步梳理 17分钟

教师示范朗读关键段落，学生跟读并标记三个问题：

- 伏尔泰做了什么？
- 伏尔泰付出了什么？
- 伏尔泰留下了什么？

5. 文本地图 10分钟

师生共同画出五点地图：

- 历史定位：巨星陨落，一个世纪，思想领袖。
- 社会批判：人民无知，宗教不宽容，司法无公正。
- 斗争方式：用一枝笔反对专制、迷信和不公。
- 精神核心：“微笑”中有理性、仁慈和宽容。
- 高潮召唤：让光明从坟墓里出来。

### 课堂产出

学生用自己的话写3句话概括课文，每句话必须使用一个本课词汇。

### 课后任务

复习核心词汇，并完成“文本地图”标注：在课文中用不同颜色标出“笔、微笑、光明、黑夜、斗争”等关键词或相关句子。

## 第二课：读深课文，语言与修辞

### 教学目标

- 理解雨果的演说语言风格。
- 分析排比、对比、呼告、反复等修辞手法的效果。
- 重点细读“微笑”和“光明”意象，感受文章的文学力量。

### 教学流程

1. 词汇小测 10分钟

默写10个高频难词：睿智、肃然起敬、轻蔑、讽刺、暴虐、启蒙、诅咒、呐喊、虐待、嘲笑。  
互批互改，错字旁边写一次正确字形。

2. 修辞分析 15分钟

学生找出排比、对比、呼告、反复的例子。讨论问题：

- 雨果为什么不直接说“伏尔泰很伟大”？
- 修辞怎样让观点更有节奏、更有情感、更容易打动听众？

教师总结：修辞不是装饰，而是观点获得力量的方式。

3. 核心意象细读 20分钟

分组研读四个意象：

- 巨星陨落：把伏尔泰写成历史性人物。
- 一枝笔：把思想写成武器。
- 微笑：对强者是嘲笑，对弱者是安抚。
- 光明：把启蒙、真理、正义和仁慈写成可见的力量。

重点细读第10至12段：“微笑，就是伏尔泰。”  
引导学生回答：这微笑为什么不是普通的笑？它怎样同时包含讽刺、智慧、怜悯和宽容？

4. 演说与普通议论文比较 10分钟

比较：

- 演说：面对听众，情感更强，句式更有节奏，适合朗读。
- 普通议论文：面对读者，论证更稳，语气更平实，适合细读。

5. 小组朗读 5分钟

每组选一句或一小段，处理重音、停顿和情绪。教师提醒学生注意括号中的“鼓掌”“喝彩”“欢呼”，理解现场反应如何证明演说的感染力。

### 课后任务

从课文中抄出最有力量的一句话，写两三句解释：它有力量，是因为观点深刻、修辞有力，还是因为它制造了演说现场感？

## 第三课：议论文结构训练，以课文为支架

### 教学目标

- 学习议论文的“论点、论据、论证”结构。
- 理解雨果的论证并不只是列事实，而是事实、对比、价值判断和修辞气势共同完成。
- 完成一段有论点、有例子、有分析的议论段落。

### 教学流程

1. 词汇复习 5分钟

抽查15个中等难度词，要求学生口头造句。

2. 议论文结构拆解 20分钟

围绕问题：“雨果怎样证明伏尔泰代表人类良知？”

师生共同填写结构表：

- 中心论点：伏尔泰代表思想自由与人类良知。
- 事实论据：他为西尔旺、蒙巴伊、卡拉斯、拉巴尔等受害者斗争。
- 对比论证：人民/权贵，弱者/强者，理智/偏见，正义/非正义。
- 价值判断：思想自由、理性、宽容高于暴力和迷信。
- 修辞气势：排比、呼告、反复让论证变成现场召唤。

3. 写作框架训练 10分钟

板书一段议论的基本结构：

- 论点：我想证明什么？
- 论据：我凭什么这样说？
- 分析：这个例子怎样支持观点？
- 提升：这个观点对今天有什么意义？

4. 课堂仿写 20分钟

题目：“一个敢说真话的人，对社会有什么价值？”  
要求：

- 150字左右。
- 有清楚论点。
- 至少1个具体例子。
- 至少使用3个本课词汇。
- 不能只写故事梗概，必须表达判断和分析。

5. 同伴互评 5分钟

检查三个问题：

- 论点清楚吗？
- 论据能支撑论点吗？
- 有没有分析，而不只是举例？

### 课后任务

把课堂段落扩展成300字完整议论文初稿。至少加入一个现实观察，作为第二个论据。

## 第四课：现实连接，批判性思维与反思

### 教学目标

- 将课文主题与学生经验连接。
- 理解现代社会中的“启蒙”可以表现为信息素养、独立判断和责任表达。
- 区分理性批判与情绪宣泄。

### 教学流程

1. 热身讨论 10分钟

展示两个情境，学生选择立场并说明理由：

- 班上有同学指出考试制度不公平，但大家都不敢附和，你会怎么做？
- 社交媒体上一个观点很流行，但你认为它是错的，你会公开反驳吗？

可选立场：支持、沉默、私下提醒、公开反驳。  
教师追问：你的判断依据是什么？你愿意承担什么代价？你如何避免伤害无辜的人？

2. 主题深化讨论 15分钟

讨论：“伏尔泰的启蒙放在今天可能是什么？”

可能方向：

- 信息素养
- 批判性思维
- 走出信息茧房
- 拒绝网络暴力
- 不盲从权威
- 尊重证据与事实

要求学生尽量使用：启蒙、批判、权威、专制、理性、照亮。

3. 批判的边界 15分钟

教师引导：

- 证据：先确认事实，避免把偏见包装成勇敢。
- 理性：反对权威时，也要能解释自己的理由。
- 宽容：批判错误，不把不同意见的人简单妖魔化。

总结句：今天学习伏尔泰，不是学会更响地反驳，而是学会更清醒、更负责地判断。

4. 个人反思写作 15分钟

题目：“如果你是伏尔泰，你愿意付出这些代价吗？为什么？”  
要求：

- 不只写“愿意”或“不愿意”。
- 写出真实顾虑。
- 写出价值判断。
- 至少使用2个本课核心词。

可用句式：

“我愿意/不愿意，因为我担心……但我认为……”

5. 分享与讨论 5分钟

2至3位学生自愿分享。教师强调：没有标准答案，重要的是思考过程是否诚实、清楚、有依据。

### 课后任务

修改议论文初稿，加入自己的现实观察，并检查是否做到“观点清楚、例子具体、分析充分”。

## 第五课：词语填空测试，写作定稿与单元总结

### 教学目标

- 完成词汇理解与应用考核。
- 完成议论文最终修改。
- 总结本文关于语言、思想和人的尊严的启示。

### 教学流程

1. 词语填空测试 25分钟

25题填空测试，提供词库辅助。  
提醒：

- 先看句意，再看拼音。
- 不照词库顺序抄。
- 四字词语必须写完整。
- 不确定时回到课文语境判断。

目标是考查理解和应用，不是机械背诵。

2. 写作互评与定稿 20分钟

同伴用四个问题互评：

- 论点清晰吗？
- 论据有说服力吗？
- 分析是否真正解释了论据和论点的关系？
- 有没有自己的声音和现实观察？

学生根据反馈修改并提交最终版。

3. 单元总结 15分钟

教师总结：

- 一枝笔：思想可以成为反抗不公的武器。
- 一个微笑：理性批判也可以带着仁慈。
- 一束光：真正的语言会照亮后来的人。

学生完成一句话：

“读完这篇课文，我重新思考了______。”

### 课后延伸

可选任务：选择“笔、微笑、光明”中的一个意象，写一段100字短评，说明它为什么能代表本文的精神。

## 评估概览

| 评估项目 | 课时 | 建议权重 | 评估重点 |
| --- | --- | ---: | --- |
| 词语填空测试 | 第5课 | 25% | 词义理解、语境判断、字形准确 |
| 课堂参与 | 第2、4课 | 20% | 朗读投入、讨论质量、倾听与回应 |
| 议论文写作 | 第3至5课 | 40% | 论点、论据、分析、语言准确、现实观察 |
| 个人反思短文 | 第4课 | 15% | 真实判断、理由清楚、价值思考 |

## 贯穿全程的词汇策略

词汇不单独用一节课教完，而是螺旋式出现：

- 第1课：整体预习，标记不懂的词。
- 第2课：在语境中理解，分析词语在修辞和意象中的作用。
- 第3课：在写作中主动运用。
- 第4课：在讨论和反思中自然使用。
- 第5课：通过测试和定稿巩固。

这样学生接触每个核心词至少4次，从“认识”走向“会用”。

## 教师提醒

- 不要把本文只教成“勇敢说真话”。更准确的核心是：用理性、良知和宽容反抗专制、迷信与不公。
- 不要把修辞只当作答题术语。本文的排比、对比、呼告、反复是演说力量的一部分。
- 不要跳过“微笑”。这是全文最有文学价值的细节，也是学生理解伏尔泰精神的入口。
- 现实连接要保持开放，但也要有边界：鼓励独立判断，同时强调证据、理性和责任。
