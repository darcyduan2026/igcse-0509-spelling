# 文言文学生自学批注海报规格

## 目标

将任意一段文言文制作成一张适合学生自学、教师投屏或打印的批注海报。海报应帮助学生同时解决三件事：

1. 会读：生僻字、异读字、人名地名读音清楚。
2. 会懂：核心词语、古今异义、文言实词和难句含义清楚。
3. 会抓主旨：通过核心词汇、人物典故和结构关系理解文章观点。

用户只需提供文言文文本，不需要额外解释版式或批注要求。

## 输出文件

每次生成至少输出三个文件：

1. `文章标题_批注海报.html`
2. `文章标题_批注海报.png`
3. `文章标题_批注海报.pdf`

如果用户要求 Word 版，可额外输出：

4. `文章标题_学生自学批注版.docx`

## 基本版式

采用竖版海报布局，优先适合 A3 打印，也适合课堂投屏。

推荐尺寸：

```text
宽：1600px
高：2300px 至 2600px
```

如文本较长，可适当增高，不要压缩字体到难以阅读。如文本很短，尤其是绝句、律诗等短诗，可使用 950px 至 1800px 的紧凑画布，避免正文和批注集中在上半部分、下方留下大面积空白。

页面结构：

1. 顶部标题区
2. 颜色图例区
3. 核心词汇区
4. 中央正文区
5. 左右两侧批注区
6. 连接线层
7. 底部学习提示区

顶部标题区必须使用统一格式：`[朝代] 作者 《文章名称》 批注海报`。例如：`[明代] 宋濂 《记李歌》 批注海报`。

## 颜色规则

必须全篇统一使用颜色区分批注功能：

```text
黑色：原文正文
蓝色：拼音、读音提示
红色：字义、古今异义、文言词义、句意提示
金色/棕色下划线：人物、地名
浅蓝批注框：读音或背景制度类说明
浅红批注框：古今异义、价值判断、核心概念说明
浅米色批注框：人物典故说明
```

推荐颜色：

```text
正文黑：#222831
拼音蓝：#1d5fbf
解释红：#c63d32
人物金：#b8791a
背景纸色：#f8f3e8
批注框底色：#fffaf0
读音框底色：#eef5ff
解释框底色：#fff0ed
```

## 字体与字号

正文应清楚、稳定、有传统文本感。

推荐字体：

```text
正文：Songti SC, STSong, Noto Serif CJK SC, serif
批注：PingFang SC, Noto Sans CJK SC, sans-serif
```

推荐字号：

```text
标题：48px 至 60px
正文：28px 至 34px
拼音：正文的 0.70 至 0.75 倍
红色解释：正文的 0.70 至 0.75 倍
批注标题：22px 至 26px
批注正文：16px 至 20px
```

不要为了放下所有内容而把正文缩得过小。宁可增加海报高度。

## 批注层级

批注必须分层，不要把所有词都解释一遍。优先标注对学生理解最有帮助的内容。

### 第一层：核心词汇

海报顶部必须有一个“核心词汇”区，列出 3 至 6 个最能支撑主旨的词语。

每个核心词卡片包含：

```text
词语
一句学生能懂的解释
```

例如：

```text
四民：士、农、工、商，是全文讨论的四种职业。
异业 / 同道：职业分工不同；价值归向相同。
尽心：把自己的职责做到充分，并能益人。
归要：最后归结的要点。
```

选择核心词时优先考虑：

```text
论点词
结构词
价值判断词
反复出现的关键词
与主旨直接相关的词
```

### 第二层：正文内嵌批注

正文中直接嵌入短批注，帮助学生读到这里就能懂。

格式：

```text
词语(拼音)
词语（解释）
词语（古义：……；今义：……）
```

规则：

1. 拼音放圆括号，用蓝色。
2. 意思放中文圆括号，用红色，例如“觇之（察看）”“生人（古义：百姓）”。
3. 一个批注尽量不超过 12 个汉字。
4. 不要给每个字都标拼音，只标生僻字、异读字、容易误读的人名地名。
5. 如果左右批注框标题或正文中提示了读音，正文对应词语后也必须同步标出蓝色拼音；读音不能只出现在旁注里。
6. 同一个字在文中有不同读音或不同义项时，应在正文对应位置分别标注，例如“年少(shào)（年轻）”与“少(shǎo)（稍微）留”。
7. 词义、古今异义、核心实词、句意提示、价值判断、结构提示都应优先保留在正文中，用红色圆括号随文解释，不放到左右批注框。
8. 导出前必须逐项核对所有旁注框里的拼音和“读……”提示，确认正文对应词语后都有蓝色拼音。

### 第三层：左右批注框

正文两边放批注框，只用于解释人物和地名，帮助学生辨认“谁”和“哪里”。普通词语、古今异义、核心实词、句意提示、结构提示和价值判断不得放在两侧批注框中，必须放回正文红色圆括号内，避免学生在正文和两侧之间来回寻找解释。

适合放入批注框的内容：

```text
人物是谁
人物在文中起什么作用
地名在哪里或指向什么空间
人物或地名背后的必要背景
```

每个批注框包含：

```text
标题
1 至 2 句解释
可选：一句学习提醒
```

示例：

```text
傅说 (fù yuè)
商代贤臣，原从事版筑劳作。
“说”作人名读 yuè，不读 shuō。
```

### 第四层：连接线

人物、地名批注框必须用细线连回正文中的对应词语。

连接线规则：

1. 左侧批注框连向正文左侧或中部词语。
2. 右侧批注框连向正文右侧或中部词语。
3. 线条要细、浅、柔和，不抢正文。
4. 人物和地名线用金色系。
5. 如果人物或地名本身有读音提示，可用蓝色系。
6. 不再为普通词义、古今异义、价值判断或结构提示设置侧边连线。
7. 连接线不得遮挡正文阅读。

推荐线条：

```text
stroke-width: 2px 至 2.5px
opacity: 0.45 至 0.60
曲线连接，避免硬折线
```

## 需要自动识别并优先批注的内容

### 生僻字和异读字

标注拼音，放在字或词后面，用蓝色。

所有拼音提示都必须优先落实到正文对应词语上。左右批注框可以重复提示读音，但不得出现“旁注有拼音、正文无拼音”的情况。

优先标注：

```text
生僻字
多音字
人名异读
地名异读
常见误读字
```

例如：

```text
傅说(fù yuè)
荷蒉(hè kuì)
斫(zhuó)
工贾(gōng gǔ)
```

### 古今异义

必须用红色解释。

格式：

```text
生人（古义：百姓，不是“陌生人”）
```

如果解释较长，正文中仍写短注；不要把普通词义说明移到侧边批注框。

### 核心实词

必须批注字面义或语境义。

例如：

```text
修治（治理、教化）
具养（供养民生）
通货（使货物流通）
归要（最终要点）
```

### 人物与典故

必须识别所有人物和历史地名。人物和地名可放入左右批注框；典故、制度名、核心概念若不是人物或地名，应放入正文红色圆括号中解释。

人物批注必须回答：

```text
他是谁？
他在文中证明什么？
为什么作者要提他？
```

不要只写百科式简介。批注要服务文本理解。

例如：

```text
伊尹
商初贤相，传说曾在莘野耕作。
说明贤才可以出于农耕。
```

### 论证结构词

遇到表示论证推进的词，应适当解释。

例如：

```text
故曰：总结前文观点
盖昔：引出历史证据
自……而……：引出价值转折
于是：推出后果
```

## 海报内容生成流程

每次生成海报时按以下步骤处理文本：

1. 通读全文，确定中心论点。
2. 提取 3 至 6 个核心词汇，放入顶部核心词汇区。
3. 标出生僻字、异读字、多音字，加入蓝色拼音。
4. 标出古今异义、核心实词、难懂短语，加入红色短注。
5. 提取人物和地名，制作左右批注框；其他词义、典故、制度、结构和主旨提示全部回到正文红色圆括号中。
6. 将人物、地名批注框与正文关键词用线连接。
7. 检查海报是否能让学生完成自学：
   - 会读重点字词
   - 知道核心词语意思
   - 理解人物典故作用
   - 抓住文章主旨
   - 旁注出现的所有读音，都已逐项核对，并在正文对应词语后同步标注
8. 排版自检：渲染后检查标题、图例、核心词汇区、正文、侧栏批注和页脚的实际位置，确认没有重叠遮挡，且底部没有大面积空白。
9. 最终自检：检查当前标出的注释是否能够帮助学生梳理发音、解决字面意思，并指导学生开始思考该篇文言文所表达的核心思想。
10. 导出 HTML、PNG、PDF。

## 版面质量要求

生成后必须检查：

1. 正文没有被批注、连接线或解释遮挡。
2. 蓝色拼音和红色解释清楚可辨。
3. 左右批注框不重叠。
4. 连接线能连回准确词语。
5. 顶部核心词汇区不遮挡左右批注；左右第一排批注框应从核心词汇区下方开始，建议至少留出 16px 至 24px 的垂直间距。
6. 旁注中的读音已全部回到正文中用蓝色拼音标注。
7. 海报底部留有学习提示，并用一句话点明能引导学生思考的核心主旨或核心问题。
8. 当前标出的注释能够同时服务三件事：梳理发音、解决字面意思、引导学生开始思考文章核心思想。
9. 海报整体上下分布均衡；短诗海报不得在正文和批注结束后留下大面积空白，应通过紧凑画布或重新分布批注解决。
10. 长文海报正文结束到页脚之间也不能留下大面积空白；一般应控制在约 80px 至 180px，或不超过海报高度的 10%。
11. 导出前必须至少检查一次渲染后的 PNG 或浏览器截图，确认没有批注压住核心词区、正文或页脚。
12. PNG 清晰，适合投屏。
13. PDF 可打印，背景颜色正常。

## Word 版本要求

如果用户要求 Word 版本，应生成学生自学讲义，而不是海报。

Word 版结构：

```text
标题
颜色说明
一、原文批注版
二、重点词语卡
三、人物典故卡
四、朗读提示
```

Word 版颜色规则同海报：

```text
蓝色：拼音
红色：意思 / 古今异义
黑色：正文
```

Word 版不强制使用连线，但必须保留人物典故说明。

## 文件命名

文件名也应尽量与海报顶部标题一致，采用：

```text
[朝代] 作者 《文章名称》 批注海报.html
[朝代] 作者 《文章名称》 批注海报.png
[朝代] 作者 《文章名称》 批注海报.pdf
```

示例：

```text
[明代] 宋濂 《记李歌》 批注海报.html
[唐代] 杜甫 《严郑公宅同咏竹》 批注海报.png
[宋代] 苏轼 《於潜僧绿筠轩》 批注海报.pdf
```

如果没有明确题目，可从文本开头提取 6 至 12 个字作为文件名。

## 最终交付说明

最终回复用户时，只需提供生成文件链接，并简要说明：

```text
已生成批注海报 PNG 和 PDF。
蓝色为拼音，红色为字义/古今异义，人物典故在两侧并用线连回正文。
```

不要长篇解释制作过程，除非用户询问。
